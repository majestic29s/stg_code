■SSH Keyの設定

ssh-keygen -t rsa -C "majestic29s@gmail.com"

■公開keyの登録
cat ~/.ssh/id_rsa.pub

■githubに接続
ssh -T git@github.com

■githubよりクローンをダウンロード
git clone git@github.com:majestic29s/stg_code.git

■コード追加

■コード登録
git add stg2130.xlsx

■登録状況確認
git status

■コミット
git commit -m "Add"

■登録状況の確認
git log

■githubのリポジトリへ登録
git push

■add commint同時実行
git commit -am "Add commit"

■リモートリポジトリへの送信
git push -u origin master

■リモートリポジトリから情報取得
git pull

■ブランチ情報を取得(github上も含む)
git buranch -a

■ブランチの差分を確認
